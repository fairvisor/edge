## [v0.1.0] - 2026-03-03

Initial public release.

### Added

- Fairvisor Edge runtime: request scoring, rate limiting, cost budgeting,
  kill switches, shadow mode, and LLM streaming support.
- CLI (`fairvisor`) for policy authoring, validation, and testing.
- Policy examples and cookbook operational guides.
- CI pipeline: luacheck, busted unit + integration tests, E2E smoke, Trivy SBOM.
- Open source governance: security policy, support policy, release process,
  PR/issue templates, and contributor docs.
