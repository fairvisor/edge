# Support

## Where to Ask

- Bug reports: GitHub Issues (use bug report template)
- Usage questions: GitHub Issues (use question template)
- Feature proposals: GitHub Issues (feature brief template)

## What to Include

- Fairvisor version or commit SHA
- Policy bundle snippet (redacted if needed)
- Reproduction steps
- Expected behavior and actual behavior
- Logs or headers relevant to the decision path

## Supported Versions

Community support is best-effort for:

- `master` branch
- latest tagged release

Older releases may receive guidance but are not guaranteed to be patched.
